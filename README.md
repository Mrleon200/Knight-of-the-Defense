# Knight of the Defense

Knight of the Defense is inspired by and based on the mechanics of Kingdom Rush 1, a popular tower defense game. This game aims to capture the strategic depth and engaging gameplay that made Kingdom Rush an iconic title in the tower defense genre.

## Overview

This is a tower defense game built with **Unity**, where players must strategically place defensive towers and manage hero units to repel waves of enemies. The game combines resource management, tactical positioning, and real-time decision-making.

## Features

- **Classic Tower Defense Mechanics**: Build and upgrade towers to defend your realm
- **Multiple Hero Units**: Deploy powerful heroes with unique abilities and playstyles
- **Diverse Upgrade Paths**: Customize your towers and heroes through various upgrade systems
- **Engaging Enemy Types**: Face different enemy archetypes with varying behaviors and strengths
- **Strategic Depth**: Plan your defenses carefully with limited resources and tower placement constraints

## Game Systems

### Tower Defense System
- Place towers along strategic paths
- Upgrade towers to increase damage, range, and special effects
- Different tower types with unique gameplay mechanics

### Hero Units
- Recruit and deploy hero characters
- Each hero has unique abilities and upgrade paths
- Heroes can be leveled up and customized

### Enemy Management
- Multiple enemy types with different behaviors
- Progressively harder waves as the game advances
- Strategic enemy positioning and movement patterns

### Resource Management
- Manage currency/gold for tower placement
- Balance offense and defense strategies
- Plan long-term progression

## Getting Started

### Prerequisites
- Unity Engine (check specific version in ProjectSettings)
- Basic understanding of tower defense game mechanics

### Installation
1. Clone this repository
2. Open the project in Unity
3. Load the main scene in `My project/`
4. Press Play to start the game

### Build Instructions
1. Go to File > Build Settings
2. Select your target platform (Windows, Mac, Linux, WebGL, etc.)
3. Click Build and Run

## Controls

- **Left Mouse Click**: Select towers and place them on the map
- **Right Mouse Click**: Cancel tower placement or sell towers
- **Spacebar**: Pause/Resume game
- *Additional controls to be documented as the project develops*
